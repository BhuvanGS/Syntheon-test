# Meet – byt-bbhy-uwz Calculator

A lightweight, light‑themed web calculator that supports basic arithmetic, keyboard input, and robust error handling.

## Features
- Addition, subtraction, multiplication, division
- Clean, minimal UI with large, easy‑to‑click buttons
- Responsive layout for desktop and mobile
- Division‑by‑zero displays an **Error** message
- Full keyboard support (numbers, operators, Enter, Backspace, Escape)
- Light theme with subtle accent colors

## Development
1. Clone the repository.
2. Open `index.html` in any modern browser.
3. No build step required – pure HTML/CSS/JS.
