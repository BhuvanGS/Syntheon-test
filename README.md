# Meet – rjt-upjy-ngx Calculator

A lightweight web calculator that supports basic arithmetic, clean UI, division‑by‑zero protection, and full keyboard interaction.

## Features
- **Addition, Subtraction, Multiplication, Division**
- **Responsive grid layout** with clearly labeled buttons
- **Read‑only display** showing the current expression/result
- **Graceful handling of division by zero** – displays `Error`
- **Keyboard support** – numbers, operators, `Enter` (equals), `Esc` (clear)

## Getting Started
1. Clone the repository.
2. Open `index.html` in any modern browser.
3. Use the mouse or your keyboard to perform calculations.

## Development
All source files are located in the project root:
- `index.html` – markup
- `styles.css` – styling
- `app.js` – calculator logic

No build step or dependencies are required.
